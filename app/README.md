# ワインエキスパート対策

iPhoneで使うJ.S.A.ワインエキスパート一次試験対策アプリです。

## できること

- 4択問題の練習
- 間違えた問題の自動保存
- 復習登録した問題の再挑戦
- 重要度A問題の優先出題
- 分野ごとの正答率確認
- 学習データをブラウザ内に保存

## 問題データ

問題はコード内の問題パックとして管理します。
ユーザー画面から問題を追加する機能はありません。

## 問題データ形式

問題データは次の形式で管理します。

```json
[
  {
    "id": "sample-001",
    "category": "フランス / ブルゴーニュ",
    "importance": "A",
    "question": "シャブリ地区で主に使われる品種は？",
    "choices": ["Chardonnay", "Riesling", "Merlot", "Gamay"],
    "answer": 0,
    "note": "シャブリは冷涼な産地で、白はChardonnayから造られます。"
  }
]
```

`answer` は正解の選択肢番号です。左から順に `0`、`1`、`2`、`3` を指定します。

PDFやOCR済みテキストから問題を増やす場合は、画面上で取り込むのではなく、コード内の問題パックとして追加します。
